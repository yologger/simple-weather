# Simple Weather

## Introduction
`S weather` is a simple weather app. This app fetches data from [OpenWeatherMap API](https://openweathermap.org/api) based on current location.

## Dependencies
* Kotlin
* Coroutine
* Jetpack Compose
* Hilt
* Android Navigation Component
* Android Material Component
* LiveData
* Android KTX
* Retrofit 2
* TedPermission
* Landscapist
* Accompanist
